# Carload
Short description and motivation.

## Usage
Add a model to dashboard:
```
rails g carload:admin <model_name>
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'carload'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install carload
```
